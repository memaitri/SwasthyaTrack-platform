# How to Mark Hostel Attendance - Guide for LS & MS

## Quick Reference

**Lady Superintendent (LS)**: Marks attendance for **female students only**
**Meal Superintendent (MS)**: Marks attendance for **male students only**

## Step-by-Step Instructions

### 1. Login to the System

1. Go to the login page
2. Enter your username and password
3. Click "Login"

### 2. Navigate to Hostel Attendance

1. From the main menu, click **"Hostel Attendance"**
2. You will see the hostel attendance page with:
   - Summary cards (Present, Checked Out, Vacation, Total)
   - Student list with action buttons

### 3. Mark Check-In

#### When to Use:
- Student arrives at hostel in the morning
- Student returns from leave/vacation
- Daily attendance recording

#### Steps:
1. Find the student in the list
2. Click the **"Check In"** button (📥 icon)
3. A dialog box will open
4. **Upload student image** (Required):
   - Click "Choose File" or drag image
   - Take photo with camera or select from files
   - Preview will show
5. **Add reason** (Optional):
   - Enter reason for check-in if needed
   - Example: "Returning from weekend leave"
6. Click **"Check In"** button to confirm
7. ✅ Success! Student is marked as present

### 4. Mark Check-Out

#### When to Use:
- Student leaves hostel temporarily
- Student goes home for weekend
- Student has permission to go out

#### Steps:
1. Find the student in the list (must be checked in first)
2. Click the **"Check Out"** button (📤 icon)
3. A dialog box will open
4. **Add reason** (Optional):
   - Enter reason for check-out
   - Example: "Going home for weekend"
5. Click **"Check Out"** button to confirm
6. ✅ Success! Student is marked as checked out

### 5. Mark Vacation

#### When to Use:
- Student going on extended leave
- Student sick and staying home
- Planned vacation/holiday

#### Steps:
1. Find the student in the list
2. Click the **🏖️ (Umbrella)** button
3. A dialog box will open
4. **Select start date**:
   - Click on "Start Date" button
   - Choose date from calendar
5. **Select end date**:
   - Click on "End Date" button
   - Choose date from calendar
6. **Add reason** (Optional):
   - Enter reason for vacation
   - Example: "Family emergency"
7. Click **"Mark Vacation"** button to confirm
8. ✅ Success! Student is marked on vacation for the date range

## Understanding the Student List

### Table Columns:

| Column | Description |
|--------|-------------|
| **Student** | Student name and class |
| **Gender** | 🟣 Female or 🔵 Male badge |
| **Status** | Present, Checked Out, On Vacation, or Not Marked |
| **Vacation Duration** | Shows vacation dates if on vacation |
| **Check In** | Time of check-in (if checked in) |
| **Check Out** | Time of check-out (if checked out) |
| **Actions** | Buttons to mark attendance |

### Status Indicators:

- ✅ **Present** (Green): Student is checked in and in hostel
- 🚪 **Checked Out** (Blue): Student has checked out temporarily
- 🏖️ **On Vacation** (Amber): Student is on extended leave
- ⚪ **Not Marked** (Gray): No attendance marked for today

## Daily Workflow

### Morning Routine (7:00 AM - 9:00 AM)

1. **Login** to the system
2. **Navigate** to Hostel Attendance
3. **Check the date** (should be today's date)
4. **Mark check-ins** for all students present:
   - Click "Check In" for each student
   - Upload student image
   - Confirm check-in
5. **Review summary** cards to ensure all present students are marked

### Evening Routine (5:00 PM - 7:00 PM)

1. **Check for check-outs**:
   - Students going home for weekend
   - Students with permission to go out
2. **Mark check-outs** as needed:
   - Click "Check Out" for each student leaving
   - Add reason if required
   - Confirm check-out

### Weekly Tasks

1. **Mark vacations** for students on extended leave
2. **Review monthly report** to check attendance patterns
3. **Verify** all students are properly marked

## Tips & Best Practices

### ✅ Do's:

1. **Mark attendance daily** - Don't skip days
2. **Upload clear images** - Ensure student face is visible
3. **Add reasons** - Helps with record-keeping
4. **Double-check** - Verify student name before marking
5. **Review summary** - Check totals match expected numbers
6. **Mark vacations in advance** - If you know student will be away

### ❌ Don'ts:

1. **Don't mark wrong gender** - System will block you (403 error)
2. **Don't skip image upload** - Required for check-in
3. **Don't mark attendance for other schools** - Only your assigned school
4. **Don't forget to check-out** - Mark when students leave
5. **Don't mark vacation for single days** - Use check-out instead

## Viewing Reports

### Daily View (Default)

- Shows today's attendance
- Change date using date picker
- See real-time status of all students

### Monthly Report

1. Click **"Monthly Report"** from view mode dropdown
2. Select month and year
3. View:
   - Present Days (total days student was present)
   - Stayed in Hostel (days without check-out)
   - Vacation Days (total vacation days)
   - Presence % (attendance percentage)

## Troubleshooting

### Problem: "Cannot see any students"

**Solution**:
- Verify you're assigned to a school
- Check if there are students in your school
- Ensure students have correct gender set
- Contact administrator if issue persists

### Problem: "Check In button not working"

**Solution**:
- Ensure you've uploaded an image (required)
- Check internet connection
- Refresh the page and try again
- Verify student is not already checked in

### Problem: "403 Forbidden error"

**Solution**:
- You're trying to mark attendance for wrong gender
- LS can only mark for female students
- MS can only mark for male students
- Verify student gender in the list

### Problem: "Student not found"

**Solution**:
- Student may not be in your school
- Student may be in different class
- Use search box to find student
- Contact administrator if student should be visible

## Gender-Based Access

### Lady Superintendent (LS)

✅ **Can Do**:
- View female students only
- Mark attendance for female students
- Check-in female students
- Check-out female students
- Mark vacation for female students

❌ **Cannot Do**:
- View male students
- Mark attendance for male students
- Any action on male students (403 Forbidden)

❌ **Cannot Do**:
